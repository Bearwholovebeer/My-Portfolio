# Portfolio - Titouan Lefebvre

Portfolio en HTML/CSS/JS vanilla avec shader WebGL.

## Deploiement

```bash
git init && git add . && git commit -m "portfolio"
git remote add origin https://github.com/Bearwholovebeer/Bearwholovebeer.github.io.git
git branch -M main && git push -u origin main
```

Settings > Pages > branche main > sauvegarder.

## Fonctionnalites

- Shader mesh gradient WebGL en fond
- Bouton FR/EN pour basculer l'interface
- Curseur custom SVG (fleche stylisee)
- Transitions de page en fondu
- Copyright dynamique
- Responsive mobile-first
